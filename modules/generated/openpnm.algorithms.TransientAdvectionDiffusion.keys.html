keys#

TransientAdvectionDiffusion.keys(mode=None)#

An overloaded version of keys that optionally accepts a mode

Parameters:

mode (str) –

If given, optionally, it controls which type of keys are returned. Options are:

mode

description

props

Returns only keys that contain numerical arrays

labels

Returns only keys that contain boolean arrays

models

Returns only keys that were generated by a pore-scale model

constants

Returns only keys are were not generated by a pore- scale model
